<task type="checkpoint:human-action">
  <name>Task 2: Create permanent WhatsApp System User token in Meta Business Manager</name>
  <files>N/A (Meta Business Manager UI + VPS backend/.env)</files>
  <action>
    This step is not automatable — it requires Meta Business Manager UI access to generate a permanent token.

    Follow these steps:
    1. Go to business.facebook.com → Business Settings → System Users
    2. Create a System User with Admin role (or select existing Admin user)
    3. Click "Generate New Token"
       - Select your WhatsApp application from the app dropdown
       - Check both: whatsapp_business_messaging AND whatsapp_business_management
       - Click "Generate Token"
       - Copy the token immediately (it will not be shown again)
    4. Go to developers.facebook.com → Your WhatsApp App → Getting Started
       - Find the "Phone Number ID" field (long numeric ID, e.g. 123456789012345)
       - Copy it (this is WHATSAPP_PHONE_NUMBER_ID)
    5. SSH to VPS and update backend/.env:
       Set WHATSAPP_ACCESS_TOKEN=&lt;permanent-system-user-token&gt;
       Set WHATSAPP_PHONE_NUMBER_ID=&lt;numeric-phone-id&gt;
    6. Run: docker compose restart backend
    7. Confirm: docker compose ps shows backend as Up

    CRITICAL — avoid these mistakes:
    - Do NOT use the developer token from "Getting Started" page (expires in 60 days)
    - Do NOT use the display phone number — use the numeric Phone Number ID
    - Env var must be WHATSAPP_ACCESS_TOKEN (not WHATSAPP_TOKEN)
  </action>
